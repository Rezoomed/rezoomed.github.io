 # rezoomed.github.io
  
  The official resource hub for [**Rezoomed**](https://www.rezoomed.com) — an AI
  resume platform that helps job seekers build ATS-friendly resumes, match them to
  any job description, and get their LinkedIn profile found by recruiters.
  
  🔗 **Live site:** https://rezoomed.github.io
  
  ## What's here
  A lightweight static page linking to Rezoomed's free tools and open-source guides.

  ## Rezoomed tools   
  - [ATS Resume Checker](https://www.rezoomed.com/ats-checker) — find what's filtering out your resume
  - [Resume Match Score](https://www.rezoomed.com/match-score) — score your fit against any job description
  - [AI Resume Builder](https://www.rezoomed.com/resume-builder) — build a recruiter-ready resume
  - [LinkedIn Profile Optimizer](https://www.rezoomed.com/linkedin-optimizer) — get found in recruiter searches
